> Real estate analytics project focused on understanding the factors that influence property pricing and investment attractiveness.

---

## Overview

This project explores how pricing, amenities, location, and other property characteristics can be analyzed to better understand market behavior and potential ROI signals.

---

## Business Problem

Real estate analysts and investors often need to answer:
- which factors are most associated with price variation
- how location changes value patterns
- which amenities or features correlate with higher pricing
- where investment opportunities may be stronger

---

## Workflow

```text
Property Data → Cleaning → EDA → Pricing / Location Analysis → Insight Visualization
```
